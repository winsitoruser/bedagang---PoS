## üîó API ENDPOINTS (For Developers)

### Authentication
- **Login:** POST `/api/auth/signin`
- **Logout:** POST `/api/auth/signout`
- **Session:** GET `/api/auth/session`

### Business Config
- **Get Config:** GET `/api/business/config`
- **Get Types:** GET `/api/business/types`

### Admin APIs
- **Tenants:** GET/POST `/api/admin/tenants`
- **Tenant Detail:** GET/PUT/DELETE `/api/admin/tenants/[id]`
- **Tenant Modules:** GET/PUT `/api/admin/tenants/[id]/modules`
- **Modules:** GET/POST `/api/admin/modules`
- **Business Types:** GET/POST `/api/admin/business-types`
- **Analytics:** GET `/api/admin/analytics/overview`

### Business APIs
- **Products:** GET/POST `/api/products`
- **Inventory:** GET/POST `/api/inventory`
- **Customers:** GET/POST `/api/customers`
- **Tables:** GET/POST `/api/tables`
- **Reservations:** GET/POST `/api/reservations`
- **Transactions:** GET/POST `/api/pos/transactions`

## üì± MOBILE ACCESS

Untuk akses dari device lain di network yang sama:

1. Cari IP address komputer Anda:
   ```bash
   ipconfig (Windows)
   ifconfig (Mac/Linux)
   ```

2. Ganti `localhost` dengan IP address:
   ```
   http://192.168.1.XXX:3001/auth/login
   ```
